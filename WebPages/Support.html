SSP Support



In this page, you will learn how to use SSP to your advantage and what to do if you encounter bugs.

How to use SSP?

This program will hopefully guide you in the right path and decide on whether business is the right think for you. Now, obviously this isn't the only thing you need for a successful start-up. You will need an expert in the field you wish to start a business. Right now in the pre-alpha version, the GUI hasn't been added yet. So you need a python interpreter to use the code. Make sure to read the README file for detailed instructions and help.

How to deal with bugs?

If you encounter a bug, check the templates folder and open the file called "bug_report". Fill it out, save it, and and drag/drop it here. If you think that the bug is extremely dangerous or if you have spotted a potential vulnerability, drop a mail. In the body section of the mail, briefly describe the bug and when you encountered it. Suggestions to fix the bug are welcome. If you have any feedback or suggestion, make sure to do the same for bugs, but use the "Feature_request" template.
