SSP Vision





Firstly, we are looking for developers fluent in python to create a GUI, (we plan on using django). If you are interested in contributing, click here. You will be redirected to the repository of SSP, where you can contribute by forking/cloning the repository locally.

What we hope to achieve

We hope to achieve the following in the shortest time possible:
  • Creating a decent GUI for users to interact with.
  • Creating an application for windows.
  • Bringing Cross Platform (MacOS and Linux)
If everything goes well, we might even start a new project to bring this application to android and iOS. Make sure to check the Support page to stay up to date and learn how to deal with bugs and other issues.

Thank you for using SSP. Wishing you the best with your start-up and I hope that this software has helped you.
