VISION FOR SSP

Firstly, we are looking for developers fluent in python to create a GUI. If you are interested in contributing, Click here. You will be redirected to the repository of SSP, where you can contribute by forking/cloning the repository locally.

We hope to achieve the following in the shortest time possible:
  • Creating a decent GUI for users to interact with.
  • Creating an application for windows.
  • Bringing Cross Platform (MacOS and Linux)
If everything goes well, we might even start a new project to bring this application to android and iOS. Make sure to check the Support page to stay up to date and learn how to deal with bugs and other issues.


Thank you for using SSP. We wish you the best with your startup. If you have any queries/issues, mail me.
